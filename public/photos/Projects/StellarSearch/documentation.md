## StellarSearch - Intelligent Message Clustering for Communities

**Timeline:** November 2025 - December 2025  
**GitHub:** https://github.com/oleksiisud/slack-cluster-finder  
**Alternative Names:** Slack Cluster Finder

### Original Description

Are you ever pissed about finding messages on Slack?

Let's admit it, modern message search algorithms are plain outdated. Linearly searching for keywords is inefficient, and frequently misses the target purpose of message searching.

The goal of Stellar Search is to move beyond simple channel analysis to deliver actionable insights by automatically identifying and grouping latent topic clusters within large, active Slack and Discord communities.

The target audiance can be anybody from large organizations to individuals. Server owners simply allow messages to be stored on the project supabase database, or can set up their own database if they wish to fully own their data. Then, users can connect to the clusters, approaching each groupchat like stellar systems in a galaxy.

Find messages, and opportunities, faster with Stellar Search!

### Additional Context

**Technology:** Latent topic modeling, semantic clustering, Slack/Discord APIs, Supabase integration  
**Use Cases:** Discover collaboration opportunities, identify experts, find historical context, understand discussion trends
