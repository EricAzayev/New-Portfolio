## Findr - Gamified Wildlife Discovery

**Timeline:** October 2025  
**GitHub:** https://github.com/BorowskiKacper/divhacks  
**Event:** DivHacks Hackathon

### Project Overview

How in touch are you with the animals that surround you? Whether you live in a dense urban setting or the quiet countryside, we often overlook the incredible wildlife sharing our space. Most people can recognize a hundred brand logos but struggle to identify the birds in their own backyard, creating a disconnect from the natural world.

That's why we made **Findr**, a gamified education app that turns every walk into a scavenger hunt. Using the Gemini API, users simply snap a photo of an animal to receive instant species identification and fascinating ecological insights. It's more than a camera; it's a digital field guide that transforms a fleeting glance into a lasting piece of knowledge.

### Core Features

- **Species Identification**: Real-time AI photo recognition using Google's Gemini API
- **Location Pinning**: Mark sightings geographically to build a community heatmap
- **Rarity-Based Ranking**: Gamified leaderboard system rewarding rare species discovery
- **Migratory Pattern Focus**: Special bonuses for spotting migratory species, teaching users about global biodiversity
- **Community Engagement**: "Flex" sightings with friends and inspire local exploration

### Technical Stack

- **Frontend**: React Native
- **AI/ML**: Gemini API (species identification), Image recognition
- **Backend**: Supabase (database, real-time sync)
- **Location Services**: Native geolocation, mapping integration
- **Gamification**: Rarity scoring algorithms, user ranking system
